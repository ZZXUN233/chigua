# 华强买瓜 — 价格行情系统设计

> 2026-06-04 | feat/next-big-thing

## 背景

"华强买瓜"是2003年《征服》经典片段走出的网络热梗，2026年凭借AI技术以"买瓜宇宙"形态再次席卷全网。核心台词"这瓜保熟吗？"成为一代人的集体记忆。

本设计在现有"吃瓜大师"测熟度应用基础上，融入华强买瓜梗，新增价格上报与行情看板功能。

## 功能概述

### 1. 价格上报（发布表单新增）

用户在测瓜后发布时，可选填写：
- **单价**（元/市斤）：placeholder "两块钱一斤？"
- **购买地点**：如"楼下水果摊"
- **自己劈瓜**：🔪 开关，表示用户亲自劈开验货

价格上报是可选的，不填也能正常发布。

### 2. 买瓜宇宙广场行情看板

广场顶部增加可折叠行情面板：
- 今日均价（元/斤）
- 生瓜率 / 熟瓜率 / 熟透率
- 劈瓜勇士占比
- 华强AI锐评（根据数据生成一句梗味点评）

### 3. 华强买瓜梗植入

- 广场标题改为"买瓜宇宙"
- AI锐评随价格+熟度动态生成
- 经典台词散布在 placeholder 和提示文案中
- 行情面板空态："你瞧瞧现在哪有瓜呀"

### 4. 卡片价格标签

每条广场瓜贴展示价格信息（如有）：
- 💴 ¥X.X/斤  🔪 自己劈的

## 数据模型

WatermelonRecord 新增字段：
- `pricePerJin: number?` — 每市斤价格
- `isSelfSplit: boolean` — 是否自己劈瓜
- `purchaseLocation: string?` — 购买地点

## API

返回今日行情聚合数据，5分钟缓存。

## 华强锐评规则

| 条件 | 锐评 |
|------|------|
| 熟瓜+低价(<3元) | "这瓜保熟！价格也实在，给你点个赞👍" |
| 熟瓜+天价(≥8元) | "What's up！这瓜皮子是金子做的？" |
| 生瓜+任何价 | "你瞧瞧这生瓜蛋子！给你机会你不中用啊" |
| 自己劈+熟 | "🔪 劈瓜侠认证！萨日朗~" |
| 均价<1元 | "这价比瓜皮子还便宜！" |
